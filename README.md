PAROL6 — это высокопроизводительный настольный роботизированный манипулятор, полностью напечатанный на 3D-принтере. При проектировании PAROL6 мы стремились сделать его максимально похожим на промышленные роботы по механической конструкции, управляющему ПО и удобству использования. Программное обеспечение управления, графический интерфейс и все STL-файлы робота — полностью открыты (open-source).

# Как собрать / Где купить?

Готовый манипулятор PAROL6 можно купить на нашем сайте: https://source-robotics.com

Если хотите собрать всё самостоятельно, делайте так:

* Закупите все компоненты по [BOM (списку деталей)](https://github.com/PCrnjak/PAROL6-Desktop-robot-arm/tree/main/BOM)
* Собирайте робота по [инструкциям по сборке](https://github.com/PCrnjak/PAROL6-Desktop-robot-arm/tree/main/Building%20instructions)
* Запускайте и настраивайте по [официальной документации](https://source-robotics.github.io/PAROL-docs/)
